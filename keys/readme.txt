
TEST (INTEGRATION) ENVIRONMENT
------------------------------

mips_pay.sandbox.khpos.hu.pub - this file contains public key of K&H payment gateway 
  for integration environment (https://pay.sandbox.khpos.hu), 
  required for response signature verification

mips_pay.sandbox.khpos.hu.cer - this file contains certificate of K&H payment gateway 
  for integration environment (https://pay.sandbox.khpos.hu), 
  required for response signature verification


PRODUCTION ENVIRONMENT
----------------------

mips_pay.khpos.hu.pub - this file contains public key of K&H payment gateway 
  for production environment (https://pay.khpos.hu), 
  required for response signature verification

mips_pay.khpos.hu.cer - this file contains certificate of K&H payment gateway 
  for production environment (https://pay.khpos.hu), 
  required for response signature verification
